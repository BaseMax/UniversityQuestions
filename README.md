## 1.1 
Write a program that shows the hello text.
*Answer(s) :*
```
#include<stdio.h>
int main()
{
	printf("Hello!");
	return 0;
}
```


```
#include<stdio.h>
int main()
{
	printf("Hello!\n");
	return 0;
}
```


## 1.2
Write a program that displays the 6-star icon.
*Answer(s) :*
```
#include<stdio.h>
int main()
{
	int n;
	n = 0 ;
	while(n<6)
	{
		printf("*");
		n = n + 1;
	}
	return 0;
}
```
